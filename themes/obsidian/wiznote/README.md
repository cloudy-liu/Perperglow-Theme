# WizNote Theme for Obsidian

基于为知笔记视觉风格提取的 Obsidian 主题。

这个主题会继续保留在仓库里，但当前重构后的主线分发与 README 都围绕 Claude 主题展开，所以 WizNote 被归类为 legacy 主题，不进入 v1 一键安装器。

## 安装

1. 将 `themes/obsidian/wiznote/theme.css` 和 `themes/obsidian/wiznote/manifest.json` 复制到 Obsidian vault 的 `.obsidian/themes/WizNote/` 目录
2. 在 Obsidian 的 `设置 -> 外观 -> 主题` 中选择 **WizNote**

## 特性

- 基于 WizNote 源码 CSS/JS 提取，而非截图估算
- 亮色 / 暗色双模式
- Markdown 渲染、编辑器排版和常用 UI 元素已适配到 Obsidian
